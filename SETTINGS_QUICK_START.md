1. **MainMenu.cs** - добавлена возможность открывать настройки
2. **SetupSettingsUI.cs** - скрипт для автоматической настройки UI
3. **Settings.cs** - уже готов (сохраняет/загружает разрешение)

## 🔧 ОДИН ШАГ, КОТОРЫЙ НУЖНО СДЕЛАТЬ В UNITY:

### Запустите автоматическую настройку:

1. Откройте **Unity Editor** с проектом GameYear
2. Откройте сцену: **Assets/Scenes/Menu.unity**
3. В верхнем меню выберите: **Tools → Setup Settings UI in Menu**
4. Дождитесь сообщения: "Settings UI создан успешно!"
5. **Сохраните сцену** (Ctrl+S)
6. **Запустите игру** (Play)

---

## 🎯 Как это будет работать:

### В главном меню:
- Есть кнопка **"Настройки"** (SettingButton)
- При нажатии открывается панель с:
  - 📐 **Выбор разрешения** - выпадающий список всех доступных разрешений
  - 🖥️ **Полноэкранный режим** - переключатель
  - ✅ **Применить** - применяет и сохраняет настройки
  - ❌ **Закрыть** - закрывает панель

### После применения:
- Разрешение экрана меняется мгновенно
- Настройки сохраняются в PlayerPrefs
- При следующем запуске игры настройки восстанавливаются

---

## 📋 Технические детали:

### Что делает SetupSettingsUI.cs:
1. Находит панель "Setting" в сцене
2. Очищает её от старых элементов
3. Создаёт новые UI элементы:
   - Заголовок "НАСТРОЙКИ"
   - Label "Разрешение экрана:"
   - Dropdown со списком всех доступных разрешений
   - Label "Полноэкранный режим:"
   - Toggle для полноэкранного режима
   - Кнопка "ПРИМЕНИТЬ" (зелёная)
   - Кнопка "ЗАКРЫТЬ" (красная)
4. Привязывает все элементы к SettingsMenu
5. Привязывает кнопку "Настройки" к MainMenu.OpenSettings()

### Как работает сохранение:
- **Settings.SetResolution()** сохраняет в PlayerPrefs:
  - `ResolutionX` - ширина
  - `ResolutionY` - высота  
  - `Fullscreen` - полноэкранный режим
- При запуске игры SettingsMenu загружает последние настройки
- Разрешение применяется автоматически

---

## 🐛 Если что-то не работает:

### Кнопка "Настройки" не открывает панель
**Проверьте:**
1. На MainMenu есть поле settingsMenu и оно привязано
2. SettingButton.onClick вызывает MainMenu.OpenSettings()

### Dropdown пустой (нет разрешений)
**Проверьте:**
1. Screen.resolutions возвращает массив (должен всегда работать)
2. SettingsMenu.Start() вызывается

### Настройки не сохраняются
**Проверьте:**
1. PlayerPrefs.Save() вызывается в Settings.SetResolution()
2. Нет ошибок в консоли Unity
